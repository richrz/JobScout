## Startup Contract For New Agents

The active startup contract lives in `current-pointer.md` under `First 10 Minutes Contract`.

Use it to ensure a new/no-context agent:
- confirms source-of-truth docs quickly
- commits to the first file and verification plan early
- ships a small concrete artifact fast instead of prolonged exploration narration
