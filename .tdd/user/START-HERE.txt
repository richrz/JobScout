## Your Workflow

### First Time Setup
Copy the contents of `A-SETUP-PROJECT.txt` to a NEW AI chat.
The agent will create your PRD and generate tasks.

### Each Coding Session
1. Copy `B-SETUP-SESSION.txt` to a NEW AI chat â†’ Agent claims next task
2. Copy `1-BUILD.txt` to CONTINUE â†’ Agent builds the feature
3. Copy `2-AUDIT.txt` to a NEW AI chat â†’ Different agent verifies work
4. Open `3-APPROVE.txt` and do the checks YOURSELF â†’ You approve

### The Golden Rule
NEVER let an AI mark a task "done". Only YOU can approve work after verifying it yourself.
