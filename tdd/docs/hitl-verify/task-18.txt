# Create Interactive Map Visualization with Mapbox GL JS (Task 18)

## Description
Build the geographic job visualization using Mapbox GL JS with color-coded markers, radius circles, density heatmap, clustering, popups, and mobile-optimized touch controls. Integrate into the application via the /map route, displaying real job data from the database and accessible via main navigation.

## Pre-requisites
- Mapbox token configured
- Job data in database

[ ] Verify map loads with correct initial view

[ ] Test radius circles display correctly
[ ] Verify clustering works (groups nearby markers)
[ ] Test popup opens on marker click
[ ] Validate heatmap toggle functionality

[ ] Test theme toggle (dark/light mode)
