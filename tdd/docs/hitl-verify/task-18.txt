# Human Verification: Task 18 - Interactive Job Map

## What You're Testing
A map that shows job locations with clickable markers, filters, and a heatmap view.

---

## SETUP (Do This First)

1. Open your terminal
2. Navigate to the project folder
3. Run: `npm run dev`
4. Wait for "Ready" message
5. Open your browser to: http://localhost:3000

---

## TEST 1: Can You Reach the Map Page?

**Steps:**
1. Look at the bottom navigation bar (on mobile) or top navigation (on desktop)
2. Find and click the "Map" link (has a pin icon üìç)

**Expected Result:**
- [ ] Page loads without error
- [ ] You see a heading that says "Geographic Distribution"
- [ ] A map appears below the heading

**If map shows "MOCK MAP PREVIEW":** This is OK - it means no Google Maps API key is configured. You can still test the controls.

---

## TEST 2: Do Job Markers Appear?

**Steps:**
1. Look at the map

**Expected Result:**
- [ ] You see colored dots/markers on the map (if jobs exist in database)
- [ ] Markers are different colors:
  - üü¢ Green = High score jobs (70%+)
  - üü° Yellow = Medium score jobs (40-70%)
  - üî¥ Red = Low score jobs (below 40%)
  - üîµ Blue = Jobs without a score

---

## TEST 3: Does the Heatmap Toggle Work?

**Steps:**
1. Find the button that says "Markers" (top left of map controls)
2. Click it

**Expected Result:**
- [ ] Button text changes to "Heatmap"
- [ ] Button shows a flame icon üî•
- [ ] Map view changes from individual markers to a heat/density visualization
- [ ] Click again - should toggle back to markers

---

## TEST 4: Do the Filters Work?

**Steps:**
1. Find and click the "Filters" button

**Expected Result:**
- [ ] A filter panel appears below the buttons
- [ ] You see three filter options:
  - Search box (for job title, company)
  - Score slider (0-100%)
  - Date dropdown (All Time, Last Week, etc.)

**Now test each filter:**

### 4a. Search Filter
1. Type a company name or job title in the search box
2. **Expected:** Job count updates (e.g., "2 of 50 jobs shown")
3. **Expected:** Map shows fewer markers

### 4b. Score Filter  
1. Drag the "Min Score" slider to 70%
2. **Expected:** Only high-scoring jobs remain visible
3. **Expected:** Job count decreases

### 4c. Date Filter
1. Select "Last Week" from the dropdown
2. **Expected:** Only recent jobs are shown
3. **Expected:** Job count updates

### 4d. Reset Filters
1. Click "Reset" link in filter panel
2. **Expected:** All filters clear, full job count returns

---

## TEST 5: Can You Click a Job Marker?

**Steps:**
1. Make sure you're in "Markers" mode (not Heatmap)
2. Click on any colored marker on the map

**Expected Result:**
- [ ] A popup/info window appears
- [ ] Popup shows: Job title, Company name, Location
- [ ] Popup shows: Score percentage
- [ ] Popup has a "View Details" link

---

## TEST 6: Mobile Touch Controls (If Testing on Phone/Tablet)

**Steps:**
1. Open the map on a mobile device or resize browser to mobile size
2. Try pinch-to-zoom on the map
3. Try dragging the map with one finger

**Expected Result:**
- [ ] Pinch zoom works smoothly
- [ ] Map pans when you drag
- [ ] Bottom navigation shows Map icon

---

## TEST 7: Direct URL Access

**Steps:**
1. In your browser address bar, type: http://localhost:3000/map
2. Press Enter

**Expected Result:**
- [ ] Map page loads directly (no 404 error)
- [ ] All features work as tested above
