# Build Resume Generation and PDF Export System (Task 20)

## Description
Create the resume tailoring interface at /resume with live preview using ProseMirror editor, integrate with LLM service for generation, implement ATS-friendly PDF export with react-pdf, and add configurable file naming templates.

## Pre-requisites
- LLM service configured

## Verification Checklist

[ ] Test resume generation with sample job + profile
[ ] Verify LLM produces valid markdown output
[ ] Test ProseMirror editor allows editing
[ ] Validate PDF export is ATS-friendly (no complex formatting)
[ ] Test file naming template with various patterns
[ ] Verify PDF downloads with correct filename
[ ] Test all exaggeration levels produce different tones
[ ] Validate resume saves to Application.resumePath
[ ] Test batch generation for multiple jobs
[ ] Verify PDF renders correctly on mobile
[ ] Test with long content (pagination works)
[ ] Verify /resume route is accessible and renders components correctly
[ ] Test full flow: Enter /resume -> Generate with LLM -> Edit -> Download
