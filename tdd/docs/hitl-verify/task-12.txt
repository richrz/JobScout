# Design and Implement Prisma Database Schema with Migrations (Task 12)

## Description
Create the complete Prisma schema matching the PRD specifications (User, Profile, Job, Application, Config models) and set up initial database migrations with proper indexes and relationships.

## Pre-requisites
- Task 11 completed
- Database running

## Verification Checklist

[ ] Run migration successfully: `npx prisma migrate dev`
[ ] Verify all tables created in PostgreSQL: `npx prisma studio`
[ ] Test CRUD operations for each model using Prisma Client
[ ] Validate relationships work (cascade deletes, foreign keys)
[ ] Check indexes are created: Query PostgreSQL system tables
[ ] Run seed script and verify data: `npx prisma db seed`
[ ] Test schema validation with invalid data (should fail gracefully)

## If Pass
Mark task as done in task-master.
